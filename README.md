## 🧠 What it does

Mosaïk watches how you work and automatically organizes your workspace. It learns from file usage, app switching, and task patterns.

**Key point:** All processing happens locally. Your data never leaves your machine.

## 💡 Example Workflow

```python
# Mosaïk detects pattern
if user.opens("VS Code") and user.recent_files(".py"):
    mosaik.create_workspace("Python Project")
    mosaik.group_files(["*.py", "*.txt", "*.md"])
    mosaik.suggest_apps(["Terminal", "Browser"])
```
